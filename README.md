## рҹ“Ң API Endpoints Overview

### рҹҘ— Menu APIs
- GET `/menu` вҶ’ Get all food items  
- GET `/menu/{id}` вҶ’ Get single item  
- POST `/menu` вҶ’ Add food item  
- PUT `/menu/{id}` вҶ’ Update food  
- DELETE `/menu/{id}` вҶ’ Delete food  

### рҹ”Қ Advanced Features
- GET `/menu/filter` вҶ’ Filter food  
- GET `/menu/search` вҶ’ Search food  
- GET `/menu/sort` вҶ’ Sort by price  
- GET `/menu/paginate` вҶ’ Pagination  

### рҹӣ’ Cart APIs
- POST `/cart/add` вҶ’ Add item to cart  
- GET `/cart` вҶ’ View cart  
- DELETE `/cart/{id}` вҶ’ Remove item  

### рҹ“Ұ Order APIs
- POST `/cart/checkout` вҶ’ Place order  
- GET `/orders` вҶ’ View all orders
